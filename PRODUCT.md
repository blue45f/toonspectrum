# ToonSpectrum — PRODUCT.md

> 웹툰·웹소설 통합 인덱스. 플랫폼 장벽을 넘어 "무엇을, 어디서, 왜 봐야 하는지" 한 곳에서 답하는 디스커버리·랭킹·리뷰 서비스.

register: product

## Product Purpose

네이버·카카오·리디·문피아·노벨피아로 흩어진 웹툰/웹소설을 **플랫폼 무관 단일 작품 DB**로 묶고,
- **통합 검색 + 태그 디스커버리**
- **크로스플랫폼 "어디서 봐" 라우터** (연재 위치 + 무료/기다무/유료 가격 비교)
- **투명 산식 다축 랭킹** (인기·급상승·평점·정주행·완결·신작)
- **신뢰 가능한 소셜 리뷰** (가변 별점 + 스포일러 토글 + 리뷰 태그)
- **원작↔웹툰↔영상화 어댑테이션 그래프**
- **취향 프로필 기반 추천 + 트렌드 데이터 대시보드**
를 제공한다. 우리는 콘텐츠를 호스팅하지 않는다. 우리는 "고르는 단계"를 소유한다.

## Users

- **주 사용자**: 10대 후반~30대, 멀티 플랫폼 독자. 동시에 3~5개 앱을 쓰며 "다음에 뭐 볼지" 고르는 데 피로를 느낀다. 표지 아트·평점·정주행 가능 여부에 민감하고, 새벽·이동 중 모바일로 가장 많이 탐색한다.
- **핵심 작업(JTBD)**: ① 입소문 들은 작품을 검색해 "어디서 무료로 볼 수 있나" 즉시 확인 ② 취향 장르/태그로 숨은 수작 발굴 ③ 평점·리뷰로 정주행 결정 ④ 웹소설 원작의 웹툰화/영상화 추적.

## Brand & Tone

- **성격**: 큐레이터형 사서(librarian-curator). 방대한 데이터를 우아하게 정리해 보여주는 "이야기의 색인". 과장 없는 전문성 + 콘텐츠를 향한 애정.
- **목소리**: 데이터는 grotesque(인덱스·수치)의 냉정함으로, 작품에 대한 감상은 serif(문학)의 온도로. 카피는 짧고 단정하게. 느낌표 남발 금지.
- **한국어 1순위**. 사용자 대상 텍스트는 한글. 라틴 타이포는 브랜드·수치·라벨에서 악센트로.

## Anti-references (이렇게는 안 한다)

- 네온 퍼플/사이버펑크 on 블랙 (웹툰앱의 첫 번째 클리셰).
- Letterboxd 틸/그린·오렌지 그대로 베끼기 (콘텐츠 리뷰앱의 두 번째 클리셰).
- 카드 그리드 무한 반복, 똑같은 패딩의 단조로움.
- 그라디언트 텍스트, 사이드 스트라이프 보더, 장식용 글래스모피즘, hero-metric 템플릿.
- 플랫폼 앱들의 "내 플랫폼 안에 가두는" 워터가든 UX. 우리는 플랫폼을 가로지른다.

## Strategic Principles

1. **표지가 주인공이다.** 다크 캔버스는 작품 아트(여기선 타이포그래픽 커버)를 빛나게 하는 무대다.
2. **모든 숫자는 근거를 보여준다.** 랭킹 산식·평점 분포·트렌드는 투명하게 시각화한다 (스펙트럼 바).
3. **플랫폼을 가로지른다.** "어디서 봐"는 항상 1급 정보. 무료/기다무/유료를 한눈에.
4. **장르는 색을 가진다.** 18개 장르를 색상환에 매핑한 스펙트럼이 탐색의 일관된 언어.
5. **친숙함 위의 독창성.** 내비/탭/검색은 예측 가능하게, 시그니처 모먼트(인덱스 넘버럴·스펙트럼·어댑테이션 그래프)에서 개성을 낸다.

## Data Governance

- 운영 카탈로그의 권위 소스는 서버 DB `catalog_snapshot`이다. `lib/data/` 파일 seed는 제거하고, `lib/server/catalog-store.ts` 서버 런타임 카탈로그는 DB 스냅샷에서만 채운다. DB 스냅샷이 없으면 빈 런타임 카탈로그로 시작한다.
- 수집은 `scripts/crawl.mjs --json --no-file` → Nest `CatalogService` → DB 스냅샷 저장 → 메모리 카탈로그 갱신 순서로 흐른다.
- 국내 전체 확장은 `lib/server/catalog-sources.ts`의 source registry가 관리한다. 현재 crawler 소스(네이버웹툰·네이버시리즈·카카오웹툰·레진)와 제휴/정책 확인이 필요한 pending 소스를 분리한다.
- 기본 수집 모드는 `CATALOG_INGEST_MODE=off`이며, 플랫폼별 약관/robots/API 정책/제휴 가능성을 확인한 뒤 `fixed`로 전환한다.
- 저장 데이터는 작품 메타데이터, 공개 수치, 플랫폼 URL, 출처/갱신 메타데이터로 제한한다. 유료 본문, 회차 이미지 바이너리, 댓글/외부 리뷰 원문, 개인정보는 저장하지 않는다.
- 모든 랭킹은 응답 메타데이터에 산식, source, reliability, snapshot/sourceVersion을 드러내 실제 공개 수치와 추정 지표를 구분한다.

> 데이터 고지: 작품 메타데이터·공개 수치는 공개 접근 가능한 소스에서 벤치마킹 목적으로 수집한다. 평가 수·분포·완독률 등 비공개 보조 지표는 추정값으로 표시한다. 리뷰는 외부 원문을 복제하지 않고 ToonSpectrum DB에 작성·저장된 사용자 리뷰만 사용한다. 운영 전 플랫폼별 약관·robots·제휴 가능성을 별도로 검토한다.
