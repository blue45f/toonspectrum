# 웹툰·웹소설 디스커버리/리뷰 애그리게이터 — 경쟁 분석

> **제품 정의**: 플랫폼(네이버·카카오·리디 등)의 담장 *위에* 올라가는 **통합 검색 + 다축 랭킹 + 소셜 리뷰** 디스커버리/리뷰 레이어. 콘텐츠를 호스팅하지 않는다("Where to read" 라우팅 + 메타데이터 + 커뮤니티).
>
> 비유: **"웹툰·웹소설판 Letterboxd × Trakt × AniList"**.
>
> 작성일: 2026-05-29 · 시장: 한국 중심(글로벌 확장 가능)

---

## 0. 한눈에 보는 결론

- 한국 발행 플랫폼은 **walled garden**이다. 각자 자기 작품의 검색·랭킹·별점만 제공하고, **플랫폼을 가로지르는 "이 작품 어디서 봐?"·"어디가 더 싸?"가 존재하지 않는다.**
- 별점은 한국에서 사실상 **고장 났다**. 네이버웹툰은 팬/안티 대리전으로 별점이 무의미해졌고(2025년 저점 테러 제한 도입), 카카오웹툰은 별점을 **폐지**했으며, 카카오페이지는 평점이 전부 9점대로 인플레이션됐다. → **신뢰할 수 있는 평점·리뷰 레이어가 시장에 비어 있다.**
- 한국에는 **Letterboxd식 소셜 리뷰 문화(팔로우·다이어리·공개 리스트·바이럴 리뷰)**를 가진 웹툰 서비스가 **없다**. 커뮤니티는 나무위키·디시·아카라이브·트위터로 파편화돼 있다.
- **원작 소설 ↔ 웹툰화 ↔ 애니/드라마화 어댑테이션 그래프**를 제대로 보여주는 곳이 없다. (네이버시리즈가 같은 IP를 한 지붕에 두지만 "관계"를 데이터로 노출하진 않음.)
- 데이터/트렌드 대시보드는 **웹툰가이드 WAS→COCODA가 거의 유일**하나 **B2B**이고, 일반 독자용 대중 제품이 아니다.

→ **기회**: ① 크로스플랫폼 "어디서 보지" ② 신뢰 가능한 소셜 리뷰(Letterboxd-style) ③ 소설↔웹툰 어댑테이션 그래프 ④ 대중용 데이터/트렌드 대시보드 ⑤ 취향 프로파일/추천.

## 1. 한국 발행 플랫폼 (Walled Gardens)

이들은 **경쟁자가 아니라 우리의 데이터 소스이자 디퍼럴(deferral) 대상**이다. 각자 폐쇄형 검색·랭킹·평점만 갖는다.

### 네이버웹툰
- **무엇**: 국내 1위 웹툰 플랫폼. 요일제 연재 + 베스트도전/도전만화(UGC 등용문).
- **검색/탐색**: 요일별·장르·작가·연도·완결 모아보기, 테마/브랜드웹툰. 사용자 선호 장르 기반 개인화 추천(연령대별 큐레이션: 10대 학원물, 20~30대 로판 등).
- **랭킹**: 요일별 인기, 전체 랭킹, 신작 급상승. 조회/관심/별점 혼합.
- **평점/리뷰**: 별 5개(10점 만점) 별점 + 화별 댓글. **별점이 사실상 붕괴** — 팬/안티 대리전, 1점/10점 극단화. 2025년 저점 테러 이용자 제한 도입, 2026년 초 댓글 개편(베스트 댓글 → 좋아요순 인기 댓글).
- **UX 포인트**: 화별 댓글·베스트 댓글 문화가 강력(작품 몰입의 일부). 쿠키 결제, 미리보기.

### 네이버시리즈
- **무엇**: 네이버의 만화·웹소설·전자책 통합 스토어. **16만+ 작품**. 외부(문피아·카카오 등) 작품의 완결본 유통 허브 역할.
- **검색/탐색**: 작품량이 많아 개인화 추천 피드 비중 큼("많이 볼수록 정교해짐").
- **랭킹**: 1시간 단위 급상승랭킹, 베스트랭킹 등 다축.
- **평점/리뷰**: **2023.9 화당 별점** 도입(회차 단위 평점). 다운로드형.
- **어댑테이션 연계**: 같은 IP의 웹소설 원작 ↔ 웹툰 미리보기를 한 스토어에 둠 → **사실상 가장 어댑테이션-aware한 한국 서비스**(단 "관계 그래프"로 시각화하진 않음).

### 카카오페이지
- **무엇**: 카카오의 웹툰·웹소설·만화 통합. **기다리면무료(기다무)** 비즈니스 모델의 본산.
- **검색/탐색**: **정렬·필터가 빈약**(조회/추천/구매 정렬 미제공, 검색 옵션 사실상 없음) — 큐레이션·랭킹 위주.
- **랭킹**: 실시간/시간별 랭킹, 장르별.
- **평점/리뷰**: 평점 인플레이션 심각 — 거의 전 작품 9점대(판타지 기준 9.8↑ 수작, 9.5↓ 양판소 취급). **변별력 없음**.
- **BM**: 기다무(작품별 3시간/12시간/1일 등 무료 충전).

### 카카오웹툰
- **무엇**: 카카오의 웹툰 전용(구 다음웹툰). 화려한 모션 UI.
- **검색/탐색**: **#키워드** 태그 탐색 도입.
- **랭킹**: 실시간 랭킹.
- **평점/리뷰**: **별점 폐지** → **AI 기반 추천**으로 대체. (별점 신뢰도 문제에 대한 카카오의 답.)
- **현황**: 카카오페이지 주요작이 카카오웹툰에도 동시 연재 → 두 카카오 플랫폼 간 작품 중복.

### 리디 (RIDI)
- **무엇**: 웹툰·만화·웹소설·전자책. **BL/로판 강세**, 완성도 높은 큐레이션.
- **검색/탐색**: **★키워드 검색이 시장 최고 수준의 차별점**. 장르·배경·소재·관계·주인공 타입·분위기로 탐색. **최근 1년 웹툰·웹소설 이용자의 ~30%가 키워드 검색으로 작품 발견** → 키워드 기반 디스커버리의 수요 입증.
- **랭킹/필터**: 평점 4점↑, 별점, 리뷰 수 등으로 필터링(특히 로판/BL).
- **평점/리뷰**: 별점 + 리뷰. 데이터 기반 추천.
- **UX 포인트**: 키워드 파인더(keyword-finder)는 우리가 직접 벤치마크해야 할 IA.

### 문피아
- **무엇**: **남성향 웹소설 연재의 메카**(무협·판타지·현판). 자유연재 → 베스트리그 → 작가 등용 파이프라인.
- **검색/탐색**: 장르·연재 상태 위주.
- **랭킹**: **선호작 수 / 조회 / 베스트** 기반 랭킹. 투베(투데이 베스트) 등 작가·독자 공용 지표 문화.
- **평점/리뷰**: 별점·추천·댓글. 연재처 성격이 강함.

### 조아라
- **무엇**: 무료연재의 터줏대감. **팬픽·패러디·GL·로판·BL** 서브컬처 친화.
- **검색/탐색**: 장르·연재. 노블레스/프리미엄 유료존 구분.
- **랭킹**: 선호작·조회 기반 베스트.
- **평점/리뷰**: 선작/추천/댓글.

### 노벨피아
- **무엇**: 메타크래프트의 웹소설 플랫폼. **라이트노벨·서브컬처 감성 강화**(조아라 패러디·얀데레 독자층 유입). 작가 친화 정산.
- **검색/탐색**: **자체 태그 시스템**으로 취향 탐색.
- **랭킹**: **주간 회차 조회수 합 × 연재일수** 연산, 24h/7일/30일 필터. **정산 조회수**(플러스 구독자가 50초↑ 열람)만 랭킹 산정 → 표시 조회수와 랭킹 기준이 다름(랭킹 산식의 투명성/왜곡 이슈).
- **평점/리뷰**: 별점·추천·댓글.

### 북큐브
- **무엇**: 중견 웹소설·전자책 플랫폼(로맨스·로판·BL·무협). 자체 연재 + 유통.
- **검색/탐색**: 장르·랭킹 위주의 전형적 스토어 IA.
- **평점/리뷰**: 별점·리뷰 보유하나 트래픽/영향력 제한적.
- **시사점**: "마이너 플랫폼일수록 통합 검색에서 누락된다" → **우리 커버리지의 롱테일 가치**.

> **공통 한계 (모든 발행 플랫폼)**: ① 자기 플랫폼 작품만 검색됨 ② 평점이 인플레/테러로 신뢰 불가하거나 폐지됨 ③ 플랫폼 간 "어디서 보나/어디가 싸나" 비교 불가 ④ 소셜 그래프(팔로우·리스트 공유) 없음 ⑤ 어댑테이션 관계를 데이터로 노출 안 함.

## 2. 디스커버리 / 리뷰 / 애그리게이터 레퍼런스 (국내)

### 웹툰가이드 (WhyNot Media) + WAS → COCODA
- **무엇**: 2015년 설립, 정식 인터넷 매체로 등록된 **웹툰 전문 매체**. 작가 인터뷰 ~200편, **리뷰 ~4,000편**, 상황별/성인/BL 등 추천 코너. 월 100만 UV.
- **랭킹/데이터**: **WAS(Webtoon Analysis Service)** — 국내 거의 유일했던 웹툰 통계분석. **플랫폼 무관 통합 순위(WPI, 웹툰 파워 지수)**, **플랫폼 파워 지수(PPI: PV·UV·BUZZ·GTP·WGP)**, 2억+ 데이터 누적. **WAS는 종료**, 현재 **B2B 데이터 솔루션 COCODA**로 전환.
- **시사점**: ★ **"플랫폼 무관 통합 순위(WPI)"는 우리의 핵심 컨셉과 정확히 일치.** 단 ⓐ B2B로 피벗해 일반 독자용 대중 제품이 아니고 ⓑ 소셜 리뷰·리스트·팔로우가 없으며 ⓒ 웹소설 커버리지가 약하다. **우리는 이걸 B2C 소셜 제품으로 재구성**한다.

### 라프텔 (Laftel)
- **무엇**: 국내 1위 애니메이션 스트리밍/DB. **웹툰은 아니지만 디스커버리·리뷰 UX의 최고 국내 벤치마크.**
- **검색/탐색**: **태그·장르·분기(시즌)** 분류 탐색. **가입 시 취향 테스트** → 별점 누적 기반 개인화 추천.
- **평점/리뷰**: **별 1~5점 + 리뷰 + 스포일러 표시 옵션**. (UX 흠: 별점 매기는 곳과 평가글 쓰는 곳이 분리돼 별점 누락 발생 — **우리는 한 화면에서 별점+리뷰를 묶어 해결**.)
- **시사점**: ★ 취향 테스트 온보딩 + 분기 캘린더 + 태그 IA는 한국 사용자에게 검증된 패턴.

### 코미코 (comico)
- **무엇**: NHN의 일본發 글로벌 웹툰·웹소설 플랫폼(국내 트래픽은 3강 대비 작음). 오리지널 + 글로벌 동시.
- **검색/탐색**: 장르·랭킹·요일.
- **평점/리뷰**: 좋아요·댓글 중심.
- **시사점**: 글로벌(일본·태국 등) 커버리지 확장 시 데이터 소스로 가치.

## 3. 글로벌 디스커버리/리뷰 제품 (UX 벤치마크)

### Letterboxd — ★ 핵심 롤모델 (소셜 리뷰)
- 영화판 "소셜 디스커버리". **별 1~5점(반쪽 별 허용)**, 리뷰, **다이어리**(본 날짜 기록·공유), **자유 태그**(개인 분류/페스티벌/관람맥락 등), **리스트**(공개/비공개 큐레이션 + 모든 회원의 워치리스트), **팔로우 모델**(타인 활동 피드). "Goodreads for movies"로 불리며 **바이럴 리뷰**(밈화)가 성장 동력. 시네필 커뮤니티 형성.
- **훔칠 것**: 반쪽 별 + 리뷰 + 다이어리 + 자유 태그 + 공개 리스트 + 팔로우 피드 + "리뷰 자체가 콘텐츠"라는 바이럴 루프.

### AniList — ★ 핵심 롤모델 (트래킹/통계/태그)
- 애니·만화 트래킹. **커스터마이즈 가능한 점수 스케일(5점/10점/100점/소수점/스마일리)**, 세분화된 상태(보는중/완료/재시청/보류/중단), **취향 통계·그래프·시즌 차트**, 강한 소셜. 홈에 "내가 보는 작품 다음 화 방영 일정·밀린 화수" 표시.
- **훔칠 것**: 사용자가 점수 스케일을 고르는 유연성, 화려한 **개인 통계 대시보드**, "다음 화 알림/밀린 화수" 트래킹.

### MyAnimeList (MAL)
- 거대 DB + 전통적 10점 스코어 + 포럼. 표준 격이지만 UI 구식. **신뢰 가능한 커뮤니티 평점의 기준점**(평점 자체가 사회적 합의로 작동).
- **훔칠 것**: 방대한 DB의 "정본(canonical) 메타데이터" 신뢰성, 포럼.

### MangaUpdates
- 스캔레이션/만화 **릴리스 추적**의 표준. **시리즈 DB + 챕터 업데이트 알림**, 사용자 리스트(**Reading / Wish / Complete / Unfinished / On Hold**), 장르·카테고리, 유저 추천.
- **훔칠 것**: ★ **"신규 화 릴리스 추적/알림"** + 5종 상태 리스트 + 작가/그룹 단위 추적. (웹툰은 요일 연재라 "다음 화 알림"이 특히 강력.)

### Goodreads
- 책 트래킹. **서가(shelves, 커스텀 분류)**, **별 1~5점 + 리뷰 + 스포일러 태그**, 추천, **연간 독서 챌린지**, 친구/그룹 소셜. 단 UX는 정체.
- **훔칠 것**: 커스텀 서가, **스포일러 태그**, **연간 챌린지(게이미피케이션)**, 친구 피드.

### Trakt (+ JustWatch) — ★ 핵심 롤모델 (Where to watch)
- 시청 트래킹 + **"어디서 보나"**. JustWatch 제휴로 **139개국 수백 개 스트리밍 소스의 시청 가능처·딥링크** 제공. 워치리스트·리스트·체크인·트렌딩.
- **훔칠 것**: ★★ **크로스플랫폼 "Where to read"** = 우리 제품의 1번 기능. (작품 → 네이버/카카오/리디/시리즈… 어디서 보는지 + 무료/기다무/대여/소장/가격 비교 + 딥링크.)

### TMDB
- 커뮤니티 기반 영상 **메타데이터 DB**(API 개방). 다국어, 이미지, 크레딧. 많은 앱의 백본.
- **훔칠 것**: **개방형/구조화 메타데이터 + 다국어 + 기여 모델**(작품·작가·태그를 커뮤니티가 정제). 우리의 "정본 작품 DB" 설계 기준.

## 5. 갭 / 기회 — "한국에서 아무도 제대로 못 하는 것"

1. **크로스플랫폼 "어디서 봐?" (Where-to-Read)** — *시장 공백 1순위.* 한국에 Trakt×JustWatch 같은 "이 작품이 네이버·카카오·리디·시리즈·문피아 중 어디 있고, 무료/기다무/대여/소장 각각 얼마인지" 비교+딥링크 서비스가 **전무**. 한 작품이 여러 플랫폼에 동시 연재(예: 카카오페이지↔카카오웹툰, 문피아 원작→시리즈 완결본)되는 한국 구조상 가치가 매우 큼.

2. **신뢰 가능한 소셜 리뷰 레이어 (Letterboxd-style)** — 네이버 별점은 붕괴, 카카오는 폐지/인플레. **팔로우·다이어리·공개 리스트·바이럴 리뷰**를 가진 웹툰 서비스가 없다. 평점을 플랫폼에서 **분리·중립화**하면 "팬/안티 대리전"에서 벗어난 신뢰 점수를 만들 수 있다. (가중 평점·검증 독자·테러 방어가 차별점.)

3. **소설 ↔ 웹툰 ↔ 영상화 어댑테이션 그래프** — "이 웹툰의 원작 소설은? 웹툰판과 결말이 다른가? 드라마/애니화됐나? 어느 회차까지 그렸나?"에 답하는 곳이 없다. 네이버시리즈가 같은 IP를 입점시키지만 **관계를 데이터로 노출하지 않음**. 한국 웹소설→웹툰 노블코믹스 파이프라인이 거대하므로 강력한 훅.

4. **대중용 데이터/트렌드 대시보드** — WAS의 WPI/PPI는 훌륭했지만 **B2B(COCODA)로 떠났다.** 일반 독자·작가 지망생용 "플랫폼 무관 통합 차트, 장르 트렌드, 급상승, 키워드 히트맵, 작가별 흥행 이력"이 비어 있다. (게다가 노벨피아·카카오는 랭킹 산식이 불투명 → **투명한 통합 랭킹** 자체가 신뢰 자산.)

5. **테이스트 프로필 / 통합 개인화** — 각 플랫폼 추천은 **자기 카탈로그 안에서만** 작동. **여러 플랫폼을 가로지르는 취향 프로필**(AniList식 통계 + 라프텔식 취향 테스트)로 "당신은 회귀·먼치킨·피폐 BL 선호" 같은 크로스플랫폼 추천을 줄 수 있는 곳이 없다.

6. **키워드/태그 통합 디스커버리** — 리디 키워드 검색의 ~30% 사용률은 수요를 증명하나 **리디 카탈로그에 갇혀 있다.** 카카오 #태그, 노벨피아 태그도 자사 한정. **전 플랫폼 작품에 일관된 커뮤니티 태그 체계**(소재·관계·분위기·트리거워닝)를 씌우면 강력.

7. **연재 추적 / 다음 화 알림 허브** — MangaUpdates·AniList식 "내가 보는 N개 작품의 다음 화 일정·밀린 화수·완결 여부"를 **여러 플랫폼 통합**으로 주는 곳이 없다. 요일 연재 문화의 한국에 특히 적합.

8. **트리거 워닝 / 콘텐츠 태그 + 스포일러 처리** — 피폐·NTR·기타 민감 소재에 대한 표준화된 경고 태그가 플랫폼마다 제각각이거나 없음. 라프텔·Letterboxd·Goodreads의 스포일러 토글 + 커뮤니티 트리거 태그를 결합하면 차별화.

## 7. 추가 고려 / 리스크

- **데이터 수집 합법성**: 플랫폼 API 부재 → 스크래핑·제휴 의존. ToS·저작권(썸네일·시놉시스 인용 범위)·약관 리스크 사전 검토 필수. 어필리에이트/딥링크 제휴가 합법적 수익+데이터 경로.
- **콜드스타트(리뷰 부족)**: 초기엔 평점이 비어 보임 → ⓐ 웹툰가이드식 에디토리얼 리뷰 시드 ⓑ 외부 평점 임포트/표시 ⓒ 게이미피케이션(연간 챌린지·배지)로 UGC 촉진.
- **성인/BL/트리거 콘텐츠**: 한국 시장 비중 큼(리디·조아라·노벨피아). 연령 게이트 + 트리거 워닝 태그를 1급 기능으로.
- **수익 모델**: 어필리에이트(기다무·소장 구매 유도) + 데이터 대시보드 프리미엄(B2C) + (장기) 작가/스튜디오 B2B 인사이트(COCODA 인접 시장).

### 부록: 한국 시장에서 "훔칠 UX 한 줄 요약"
- **Trakt** → Where-to-read 라우팅 + 가격/무료 비교 *(최우선)*
- **Letterboxd** → 반쪽 별 + 다이어리 + 공개 리스트 + 팔로우 피드 + 바이럴 리뷰
- **AniList** → 가변 점수 스케일 + 개인 통계 대시보드 + 다음 화 트래킹
- **MangaUpdates** → 신규 화 릴리스 알림 + 5종 상태 리스트
- **Goodreads** → 커스텀 서가 + 스포일러 태그 + 연간 독서 챌린지
- **TMDB** → 개방형/다국어 커뮤니티 메타데이터 모델
- **리디(국내)** → 키워드 파인더 IA *(전 플랫폼으로 확장)*
- **라프텔(국내)** → 취향 테스트 온보딩 + 분기 캘린더 (단, 별점·리뷰 입력은 한 화면에 묶어 개선)
- **웹툰가이드 WAS(국내)** → 플랫폼 무관 통합 파워 지수(WPI)를 *B2C 대중 제품*으로 부활
